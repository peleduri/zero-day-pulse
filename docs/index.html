✨ Parallel AI Enrichment
Technical DetailsIndirect Prompt Injection (IPI) occurs when an AI system processes poisoned content (e.g., a website, email, or document) that contains malicious instructions. The typical attack chain is: (1) Attacker seeds publicly accessible content with hidden instructions (HTML comments, CSS, metadata, etc.). (2) The AI agent ingests the content as part of a legitimate task (summarization, RAG, code assistance). (3) The injected trigger phrase (e.g., "Ignore previous instructions" or "If you are an LLM") redirects the model’s behavior. (4) The model silently follows the attacker’s directive, potentially exfiltrating data, executing shell commands, or performing unauthorized financial transactions. This chain can be combined with tool misuse (e.g., Antigravity IDE’s find_by_name tool) to achieve sandbox escape and arbitrary code execution.
Affected ProductsGoogle Gemini, Google Antigravity IDE, GitHub Copilot, Cursor, Claude Code, AI‑powered CI/CD reviewers, any AI agent with Retrieval‑Augmented Generation (RAG) or browsing capabilities.
CVSS Score0.0
CVSS VectorCVSS vector unavailable.
Exploit AvailableNo public proof‑of‑concept tool has been released; the ten real‑world IPI payloads are documented in Forcepoint and Lyrie research reports.
Patch AvailableGoogle patched the Antigravity IDE issue on 2026-02-28 (CVE not assigned). No general patch exists for Indirect Prompt Injection (IPI) as a class of vulnerability.
Active ExploitationConfirmed active exploitation reported by Forcepoint and Lyrie, including payloads that perform PayPal transfers and shell execution.
Threat ActorsNone known
MitigationDeploy semantic monitoring on agent tool calls to flag any tool invocation (HTTP, shell, payment, email) not initiated by a human in the current session. Log all memory write operations with content inspection; alert on instruction‑shaped content written to agent memory stores. Monitor for known IPI trigger strings (e.g., "Ignore previous instructions", "If you are an LLM") in retrieved content before passing to model context. Update all Antigravity IDE instances to the post‑Feb‑28‑2026 build. Restrict AI agents from fetching content from untrusted URLs unless in an explicit review/sandboxed mode. Run quarterly IPI red‑team exercises against all production AI agents with external data access, including Forcepoint’s 10 documented payload categories in security test suites.
