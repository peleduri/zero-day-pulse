Threat ActorsNone known
MitigationUpgrade to LiteLLM v1.83.7 or later. If upgrade is not possible, disable or block the /mcp-rest/test/connection and /mcp-rest/test/tools/list endpoints via firewall/reverse‑proxy, revoke or restrict unnecessary API keys, disable stdio transport, and set disable_error_logs: true as a temporary workaround.
