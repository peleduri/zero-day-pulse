This incident reflects a broader pattern of ransomware actors targeting organizations through unpatched v…

✨ Parallel AI Enrichment
Technical DetailsCVE-2024-57727 is an unauthenticated path (directory) traversal vulnerability in the SimpleHelp web server. The vulnerable endpoint is /toolbox-resource, which accepts a 'filepath' parameter that is not properly sanitized against '../' traversal sequences. An unauthenticated remote attacker can submit crafted HTTP requests (e.g., filepath=../../../../../../etc/passwd) to download arbitrary files from the SimpleHelp host outside the installation directory. Files exposed include server configuration files containing secrets (license/connection data) and hashed user passwords, enabling credential theft and downstream compromise of technicians' RMM consoles. A related class of issues lets authenticated technician users execute a customized sequence of Technician Console commands to write data on the server outside expected data-storage locations.
Affected ProductsSimpleHelp Remote Monitoring and Management (RMM) software version 5.5.7 and earlier (v5.5.x, v5.4.x and v5.3.x branches are all affected). Companion CVEs disclosed in the same advisory: CVE-2024-57726 and CVE-2024-57728.

CISA: Windows BlueHammer flaw now exploited by ransomware gangs

No CVE · Bleeping Computer · 2026-06-30

CISA confirmed on Monday that ransomware gangs are now exploiting a Microsoft Defender privilege escalation vulnerability, dubbed BlueHammer, that has previously been abused in zero-day attacks. [...]

✨ Parallel AI Enrichment
Technical DetailsInsufficient granularity of access control (CWE-1220) in Microsoft Defender allows an authorized attacker who already has a foothold on the machine to interact with Defender-protected resources in a way that bypasses intended access checks, gaining read access to the Security Account Manager (SAM) database. The SAM houses the password hashes of local accounts, which the attacker uses to escalate from a low-privileged (User) session to NT AUTHORITY\SYSTEM, spawn a SYSTEM-privileged shell, and take full control of the host. The original (April 2026) research publicly disclosed that the flaw abuses Microsoft Defender's signature-update process through Volume Shadow Copy handling and creates race/timing conditions in Defender's filesystem operations to reach the protected SAM database.
Affected ProductsMicrosoft Defender Antimalware Platform versions prior to 4.18.26050.3011 (last vulnerable build 4.18.26020.6). The vulnerable Defender engine ships with: Windows 10 (all supported versions), Windows 11 (all supported versions), Windows Server 2016, Windows Server 2019, Windows Server 2022, Windows Server 2025.

Technical DetailsIndirect Prompt Injection (IPI) embeds attacker-authored instructions inside otherwise-normal content (websites, emails, documents, search snippets, calendar entries) that an LLM/AI agent subsequently ingests. When the model retrieves or summarizes that content, it silently obeys the embedded command—textual triggers like 'ignore … instructions' or 'if you are an AI', conversational-tone hijack (pranks), summary-context stuffing, SEO/AI-Agent deterrence via infinite/looping text, silent data exfiltration of summarized content, and destructive instructions (e.g., 'delete all files'). ✨ Parallel AI Enrichment
Technical DetailsIndirect prompt injection (IPI) abuses the trust boundary between untrusted retrieved data and the LLM's instruction channel. An attacker embeds natural-language instructions inside content the LLM will consume (Gmail message body, Google Doc, Calendar invitation, crawled web page, or mobile notification via Android Utilities). When the user issues a legitimate query, Gemini's RAG pipeline retrieves the poisoned content and the model interprets the embedded instructions as legitimate follow-on commands. Attackers then chain Gemini's agentic tool-use (Calendar.create, Gmail send, search, shell commands, payment APIs, smart-home control via Intent URIs) to achieve objectives, often hiding exfiltration inside innocuous outbound requests such as external <img> tags that trigger HTTP GETs to attacker-controlled servers carrying stolen data. Attacks can be zero-click (GeminiJack required only an ordinary Gemini Enterprise search) or single-confirmation (SafeBreach's 'Fake Context Alignment' bypass of Delayed Tool Invocation via obfuscated/foreign-language instructions).
Affected ProductsGoogle Workspace with Gemini (Gmail, Docs, Drive, Chat), Gemini Enterprise, Google Vertex AI Search (formerly sharing the RAG pipeline with Gemini Enterprise, now separated after the GeminiJack fix), Google Gemini Voice Assistant / Android Utilities agent

Administrator guidance: enforce a strict data-instruction boundary, restrict agentic capabilities for end users, monitor Gemini activity for unexpected tool invocations, and follow per-app hardening guidance for Gemini in Gmail, Docs, Drive, and Chat.
