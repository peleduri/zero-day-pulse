- **Technical Details:** CVE-2025-48530 is a linear buffer overflow in CrabbyAVIF, the AVIF image decoder that Google adopted in Android using Rust. The flaw stems from incorrect bounds checking inside an `unsafe` Rust code block in the AVIF parser, producing out-of-bounds reads/writes when a malformed AVIF image is processed. - **Patch Available:** Yes. Fixed in the CrabbyAVIF source in AOSP and addressed in the Android Security Bulletin—August 2025 under security patch levels 2025-08-01 and 2025-08-05.
- **Active Exploitation:** No. Google states this near-miss Rust memory safety vulnerability was discovered internally before it ever shipped in a public Android release.
