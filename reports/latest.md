**Parallel AI Enrichment:**

- **Technical Details:** Indirect Prompt Injection (IPI) is a class of attack in which an adversary embeds malicious natural-language (or hidden) instructions inside external content that an LLM later consumes—webpages scraped by an AI browser/summarizer, documents ingested by a RAG pipeline, or emails fetched through tool-use/function-calling. When the model retrieves or summarizes that content, the injected instructions override or subvert the user's original prompt, causing the agent to call tools or generate output the user did not request. Specific techniques include: payload splitting across documents (e.g., a resume), multimodal injection hidden in images, obfuscation (Base64, emoji, multi-language), adversarial suffixes, hidden text (white-on-white, visually-hidden CSS classes), code injection in function-calling outputs, and cross-document exfiltration where one document instructs Gemini to forward data from another. No direct user input required beyond the normal query that pulls untrusted content.
